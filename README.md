# WhyNotWorking

WhyNotWorking is a Stackoverflow clone project.

## Installation

Use the package manager [yarn](https://yarnpkg.com/) or [npm](https://www.npmjs.com/) to install foobar.

```bash
npm install
yarn install
```

## Usage

```bash
npm start
yarn start
```
